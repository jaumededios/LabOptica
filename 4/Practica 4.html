Ajust contra les funcions del guió¶

Definim les funcions $\delta(\lambda)$ i $\lambda(\delta)$ tal i com ens les dona el gui (aproximaci de hartman):

\begin{equation} \delta (\lambda) = \delta _0 + \frac {c}{\lambda-\lambda_0} \end{equation}\begin{equation} \lambda (\delta) = \lambda_0 + \frac {c}{\delta-\delta_0} \end{equation}
In [10]:
delta_hartman = lambda lam, delta_0, c, lam_0 : delta_0 + c/(lam-lam_0)
lam_hartman = lambda delta, delta_0, c, lam_0 : lam_0 + c/(delta-delta_0)

Ara, l'objectiu seria utilitzar el mercuri com a calibratge per els parmetres de la funci de hartman. Aix podrem trobar les longituds d'ona del Cadmi, i desprs comprovar la qualitat dels resultats obtinguts. La forma natural de fer-ho seria mitjanant una regressi de mnims quadrats, caculant els estimadors associats a les desviacions estndard de cada parmetre. Tot i aix, el gui ens recomana un procediment diferent. Provarem tots dos procediments.

Mètode del guió:¶

El gui ens proposa utilitzar les segents formules per a calcular els parmetres de hartman noms utilitzant tres punts. Ens recomanen agafar la ralla groga, verda i blava:

In [16]:
l_y , l_g, l_b = hg_lambda[(0,2,4),]
d_y , d_g, d_b = hg_delta[(0,2,4),]
print l_y, l_g, l_b

5790.0 5460.7 4358.3

A partir d'aquestes ens indica, per a calcular els valors dels parmetres, que calculem abans tres valors intermedis:

In [17]:
A = (d_y-d_g)/(l_g-l_y)
B = (d_y-d_b)/(l_b-l_y)
K = (d_g-d_b)/(A-B) #K es C/(d_y-del0)

I a partir d'aquests parmetres podem calcular:

In [18]:
lam0 = l_y - K
del0 = d_y - (l_g-lam0)*A
c = K*(d_y-del0)
hartman_params_interp = np.array((del0,c,lam0))
In [19]:
r = [Latex('${:L}$'.format(x)) for x in hartman_params_hg]
r1 = [Latex('${:L}$'.format(x)) for x in (del0, c,lam0)]
r2 = ["{:0.10}".format(x.n) for x in ( del0, c, lam0)]

Càlcul de l'Index de refracció¶

Ara podem calcular ja la famosa grfica $n(\lambda)$, tant a partir dels punts calculats, com a partir de la interpolaci feta. La equaci que relaciona els dos termes s:

\begin{equation} n(\delta) = \frac {\sin\left ( \frac{\alpha +\delta}{2} \right)} {\sin\left ( \frac \alpha 2\right)} \end{equation}

Passem per tant a calcular-ho:

In [26]:
def n(delta, alpha): return unp.sin((delta+alpha)/2.)/unp.sin(alpha/2.)
In [27]:
lamb = np.linspace(4000,6500)

interp_n = n(interp_delta/360*2*np.pi, np.pi/3.)
hg_n = n(hg_delta/360*2*np.pi, np.pi/3.)

Càlcul del nombre d'Abbe¶

Ara podem passar a calcular el parmetre d'Abbe, que en teoria ens hauria de dir coses interessants sobre quin tipus de vidre tenim. La equaci del parmetre d'Abbe s:

\begin{equation} \nu_{abbe} = \frac {n_{groc}-1} {n_{blau}-n_{vermell}} \end{equation}
In [29]:
def nu_abbe (ny, nb, nr): return (ny-1)/(nb-nr)

ens cal, per una definici precisa de quines sn les longituds d'ona adequades per al "vermell", "blau" i "verd" comentats al gui. Una petita cerca a la literatura [1] ens mostra que de fet la equaci mereix ser escrita com:

\begin{equation} \nu_{abbe} = \frac {n_{D}-1} {n_{F}-n_{C}} \end{equation}

on $D$, $F$, i $C$ indiquen frequncies conegudes, concretament:

De fet, no sn les niques definicions trobades en la literatura [1]. Entre d'altres, podem trobar el subndex $D$ canviat pels subndexs $d$ o $e$, variant lleugerament els valors de la frequncia associada.

Els valors de les longituds d'ona per als subndexs utilitzats sn:

  • $\lambda_D$: 5893 $\mathring A$

  • $\lambda_F$: 4861.3 $\mathring A$

  • $\lambda_C$: 6562.7 $\mathring A$

In [30]:
LD = 5893
LC = 6562.7
LF = 4861.3
############

nC = n(delta_hartman(LC, *hartman_params)*2*np.pi/360, np.pi/3)
nF = n(delta_hartman(LF, *hartman_params)*2*np.pi/360, np.pi/3)

  • $n_C$: $1.83255 \pm 0.00015$ $\mathring A$
